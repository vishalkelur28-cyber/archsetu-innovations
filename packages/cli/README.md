# @archsetu/cli

Analyze any local codebase from the terminal. No AI, nothing sent over the network —
everything runs on your machine.

```sh
npx @archsetu/cli analyze ./my-project
npx @archsetu/cli health ./my-project
npx @archsetu/cli dead-code ./my-project
```

Add `--json` to any command for machine-readable output.

Full docs are in the
[main repo README](https://github.com/vishalkelur28-cyber/archsetu-innovations#readme).

Want to use the engine directly in your own code instead? See
[`@archsetu/core`](https://www.npmjs.com/package/@archsetu/core).
